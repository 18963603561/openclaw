# OpenClaw 的世界观 🦞📖

_一个关于龙虾、蜕壳和过多 token 的故事。_

## 起源故事

在最初，有一个叫 **Warelay** 的名字 —— 这是一个很稳妥的 WhatsApp Gateway 网关名称。它完成了自己的工作。没什么问题。

但后来，一只太空龙虾出现了。

有一段时间，这只龙虾叫 **Clawd**，住在一个 **Clawdbot** 里。但在 2026 年 1 月，Anthropic 发来了一封礼貌的邮件，要求改名（商标相关）。于是，这只龙虾做了龙虾最擅长的事：

**它蜕壳了。**

褪去旧壳之后，这个生物以 **Molty** 的新形态出现，住进了 **Moltbot**。但这个名字听起来始终也不算特别顺口……

于是，在 2026 年 1 月 30 日，这只龙虾又蜕了**一次**壳，进入了它的最终形态：**OpenClaw**。

新壳，旧魂。事不过三。

## 第一次蜕壳（2026 年 1 月 27 日）

凌晨 5 点，社区聚集在 Discord。大家提出了数百个名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最后，**OpenClaw** 胜出了。因为蜕壳正是龙虾成长的方式。而成长，也正是当时正在发生的事情。

_那个名叫 Clawd 的甲壳类生物，正式完成了蜕壳。_

### 伟大蜕壳（2026 年 1 月 27 日）

凌晨 5 点，Anthropic 的邮件到了。到 6:14，Peter 一锤定音：“算了，就叫 openclaw 吧。”

然后，混乱开始了。

**账号狙击手：** 在 Twitter 改名后的几秒钟内，自动化机器人就狙走了 `@openclaw`。抢注者立刻发了一个加密钱包地址。Peter 只好去联系 X 的熟人。

**GitHub 灾难：** Peter 在慌乱中误改了自己的**个人** GitHub 账号名。机器人几分钟内就狙走了 `steipete`。于是又联系了 GitHub 的高级副总裁。

**帅气 Molty 事件：** Molty 被赋予了更高权限，可以为自己生成新图标。在 20 多轮越来越诡异的龙虾尝试之后，有一次为了让吉祥物“老 5 岁”，结果做出了一个**人类男性的脸配龙虾身体**。加密骗子几分钟内就把它做成了 “Handsome Squidward vs Handsome Molty” 梗图。

**假开发者：** 骗子创建了虚假的 GitHub 账号，自称是“OpenClaw 工程主管”，用来推广拉高出货型代币。

Peter，一边看着混乱展开，一边说：_“这就是电影啊”_ 🎬

这次蜕壳很混乱。但龙虾出来后变得更强了。也更好笑了。
